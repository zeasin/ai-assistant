📖 老齐AI个人助理

一站式个人工作台，集成 AI 综合日报、对话问答、笔记管理、公众号运营分析、客户管理等功能。 所有 AI 能力基于本地笔记库，无需联网第三方知识库。

🚀 快速启动

本地运行（Java 版）

# 1. 编译
mvn clean package -DskipTests

# 2. 运行
java -jar target/assistant-v2-2.0.0.jar

# 服务默认运行在 http://localhost:6789

环境要求

依赖版本用途
Java≥21Web 服务运行环境

系统内置 Spring @Scheduled 驱动以下定时任务（时区：Asia/Shanghai）：

周二发文提醒每周二 09:00提醒发布「码农老齐」公众号

所有定时任务可在 配置页 手动触发执行。

📊 综合日报

首页展示 AI 综合日报，数据源为笔记库中的记忆文件，涵盖：项目进展、客户动态、今日 TODO、昨日回顾、学习方向、生活提醒、个人目标、AI 建议。

💬 AI 对话

与 AI 助理对话，它会读取整个笔记库来回答问题。支持流式响应、实时状态显示、历史会话管理。

  • 流式响应 — 回答逐字显示，无需等待全部生成完毕
  • 实时状态 — AI 读取笔记时显示 📖 文件名，知道它在做什么
  • 历史会话 — 左侧边栏保存所有对话记录，可切换、可删除
  • 自动保存 — 每次对话自动保存到 chat_sessions.json

📝 快速数据录入

核心原则：规则由你在 AI 中制定，AI 自己保存数据。 Java 代码只做消息转发，不参与任何业务逻辑。

你只需在笔记库的 AGENTS.md 中定义录入规则（字段映射、计算方式、保存格式等）， 所有入口的消息都会原样发给 AI，AI 按你的规则解析并直接写入数据文件。

入口方式说明
页面表单运营/客户/工作页面点按钮弹窗填写字段拼成文本发给 AI，AI 按规则保存
飞书消息以"记录/新增/添加/创建/录入"开头发消息原样转发 AI，AI 按规则保存并回复
AI对话聊天框以"记录/新增/添加/创建/录入"开头输入原样转发 AI，AI 按规则保存并回复

示例消息（AI 按 AGENTS.md 规则解析）

记录日报：今天修复了导出bug，下午跟进了张三的合同
记录文章：《实战指南》微信阅读3200点赞85，增粉12
新增客户：王五，公司启航科技，阶段意向，产品ERP
新增线索：赵六，来源转介绍，备注朋友推荐
记录跟进：张三 谈了合同细节，下周二再约
记录日统计：码农老齐 粉丝28500 阅读4200

📩 飞书集成

配置 Webhook URL 后，AI 综合日报、发文提醒、下班提醒等自动推送到飞书群。 在 配置页 设置 Webhook 地址。

👥 客户管理

客户管理看板展示概览、销售漏斗、合同金额、线索漏斗、近期活动、待跟进等数据。 数据存储在 企业/客户管理/ 目录。

📈 运营看板

公众号运营数据分析，数据来自 自媒体/运营数据.json。 提供各公众号粉丝数据、文章阅读量排名、平台对比、AI 自动分析内容策略效果、写作建议和选题推荐。

📂 笔记管理

内置文件浏览器，无需 Obsidian 即可管理笔记库中的 Markdown 文件：

  • 浏览 — 按目录树逐层浏览所有文件
  • 查看 — Markdown 渲染为美观的 HTML
  • 复制 — 查看页有「📋 复制文章」按钮，可直接粘贴到公众号编辑器
  • 新建 — 在当前目录创建 Markdown 文件
  • 删除 — 删除 .md 文件
