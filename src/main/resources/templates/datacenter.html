数据中心使用帮助

一、数据集管理

创建数据集：点击右上角「+ 新建数据集」，填写名称、描述，并定义 Schema 字段。

Schema 字段：每个字段包含三个属性：

  • 字段名 — 英文标识符，用于程序处理（如 title、views）
  • 类型 — 支持 文本(string)、数字(number)、日期(date)、链接(url)
  • 显示名 — 中文显示名，用于界面展示（如「标题」、「阅读量」）

编辑/删除：在数据集卡片上点击对应按钮操作。删除会永久清除所有数据。

二、数据导入

Excel 导入：

  • 支持 .xlsx 和 .xls 格式
  • 上传后会自动预览列结构，并尝试将 Excel 列名映射到 Schema 字段
  • 自动跳过空行和合并标题行，找到有效表头
  • 可在预览界面手动调整列映射后再确认导入

JSON 导入：

  • 直接粘贴 JSON 数组或对象
  • 格式：[{"字段名": "值"}, ...] 或 {"字段名": "值"}
  • 适用于从其他系统导出的结构化数据

URL 数据获取：

  • 输入目标网页 URL，AI 会自动访问并提取结构化数据
  • 如果数据集定义了 Schema，AI 会按 Schema 格式提取

三、数据浏览

选择数据集后，以表格形式展示所有记录。

  • 支持关键词搜索，在所有字段中模糊匹配
  • 每条记录显示 ID、来源标记、导入时间
  • 可逐条删除不需要的记录

四、与采集器集成

采集器任务可绑定数据集，采集完成后自动导入数据：

  1. 在数据中心创建目标数据集，记录其 ID
  2. 在采集器中编辑任务，填写「自动保存到数据集 ID」
  3. 任务执行成功后，采集结果会自动写入该数据集

五、存储说明

  • 所有数据存储在笔记库的 数据中心/ 目录下
  • 每个数据集独立目录：数据中心/{数据集ID}/
  • datasets.json — 数据集注册表
  • data.json — 该数据集的全部记录
  • imports/ — 导入历史记录
  • 每条记录自动附加 _id、_source、_importTime 元信息

六、API 接口

方法路径说明
GET/api/datacenter/datasets获取所有数据集列表
POST/api/datacenter/datasets创建新数据集
PUT/api/datacenter/datasets/{id}更新数据集
DELETE/api/datacenter/datasets/{id}删除数据集
GET/api/datacenter/datasets/{id}/records获取记录列表（支持 ?keyword= 搜索）
DELETE/api/datacenter/datasets/{id}/records/{rid}删除单条记录
POST/api/datacenter/datasets/{id}/import/excelExcel 导入（multipart/form-data）
POST/api/datacenter/datasets/{id}/import/jsonJSON 导入
POST/api/datacenter/datasets/{id}/import/urlURL 数据获取（需 AI 服务）
POST/api/datacenter/datasets/import/excel/previewExcel 预览（不导入）

导出到业务目录

AI会读取目标模块目录下的现有数据文件作为格式参考，然后将选中数据转换为相同格式后保存。
