⚙️ 定时任务（Spring @Scheduled）

由 Spring 注解驱动，替代了 crontab 和 LaunchAgent。
  • 任务描述 时间

📁 笔记库根目录配置

配置笔记库的根目录路径，修改后所有数据文件的相对路径都将基于此目录。
支持绝对路径，如 D:\notes 或 C:\Users\xxx\Notes

📁 数据文件相对路径配置

相对于笔记库根目录的路径。修改后立即生效。

🔗 飞书 Webhook 配置

修改后自动保存到 config.json，下次推送生效。

📩 飞书消息接收（API 轮询）

配置飞书自建应用的凭据，AI助理将轮询指定群的未读消息并自动回复。
先填入 App ID 和 Secret，点击"获取群列表"选择你要监听的群
填写 ERP 源码目录后，发消息含 "bug/报错/异常/排查/修复" 等关键字时自动路由到代码助理

🔧 手动触发

手动执行以下任务，立即推送飞书。
