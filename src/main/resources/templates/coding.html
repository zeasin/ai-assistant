📖 关于编程AI

编程AI 是一个基于 飞书机器人 + Pi CLI 的代码自动排查工具。你只需在飞书群中 @编程机器人 描述 bug，机器人自动调用 Pi CLI 分析项目代码，定位问题并给出修复建议。

🚀 快速开始

1
创建飞书应用

打开飞书开放平台 → 创建企业自建应用。将 App ID 和 App Secret 填入配置页。

2
配置应用权限

在应用详情 → 权限管理 → 添加 im:message 和 im:resource 权限，发布后由管理员审批。

3
配置事件订阅

在应用详情 → 事件订阅 → 添加 im.message.receive_v1 事件，推送方式选择 WebSocket。

4
启用机器人并发布

在应用详情 → 机器人 → 启用机器人能力。发布后将机器人拉入目标群。

5
配置群 ID

在配置页填入群 ID（chat_id），格式 oc_xxxxxxxxxxxx。

6
启动服务

重启 AI 助理，编程机器人 WebSocket 自动连接。即可在飞书群中 @编程机器人 开始排查代码。

❓ 常见问题

Q: 机器人没有回复？

  1. 确认 App ID/Secret 是否正确
  2. 确认已添加 im:message 权限并已发布
  3. 确认已添加 im.message.receive_v1 事件（WebSocket）
  4. 确认机器人已在群成员列表中

Q: 如何修改项目目录？

在配置页面「项目目录」输入框中直接修改，保存后重启生效。

Q: 排查结果太长被截断了？

飞书卡片消息限制 8000 字符，超出部分会被自动截断。可优化问题描述范围。
